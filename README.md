## What is Themia?

Themia is a lightweight desktop widget app for **Windows 10 & 11**. Drag, resize, and customize transparent widgets on your desktop — no browser, no Electron, fully native.

### Widgets

| Category | Widgets |
|---|---|
| **System** | CPU, RAM, GPU, Disk, Network, Battery |
| **Productivity** | Email, Calendar, Contacts, Todo, Notes, Folder |
| **Information** | Weather, Forecast, Stocks, GitHub, RSS |
| **Utilities** | Clock, Music, Color Picker |

### Highlights

- **Lightweight** — under 10 MB, barely uses any RAM
- **Fully customizable** — colors, gaps, corners, transparency
- **Multiple screens** — context-based layouts for home, work, school
- **Feels native** — drag & drop, global shortcuts, right-click menus
