## What is Themia?

Themia is a lightweight desktop widget app for **Windows 10 & 11**. Drag, resize, and customize transparent widgets on your desktop — no browser, no Electron, fully native.

### Widgets

| Category | Widgets |
|---|---|
| **System** | CPU, RAM, GPU, Disk, Network, Battery |
| **Productivity** | Email, Calendar, Contacts, Todo, Notes, Folder |
| **Information** | Weather, Forecast, Stocks, GitHub, RSS |
| **Utilities** | Clock, Music, Color Picker |

### Highlights

- **Lightweight** — under 10 MB, barely uses any RAM
- **Fully customizable** — colors, gaps, corners, transparency
- **Multiple screens** — context-based layouts for home, work, school
- **Feels native** — drag & drop, global shortcuts, right-click menus

## Roadmap

Themia is pre-1.0 and shipping fast. Here's what's on the way — order is rough, dates are not promises.

### Up next
- **More email & calendar providers** — beyond the current set
- **Widget marketplace / theme gallery** — share and install layouts
- **Cloud sync** — keep your screens and widgets in sync across devices
- **Performance pass** — lower idle CPU and faster cold start
- **Accessibility** — keyboard navigation, screen reader labels, high-contrast themes

### On the horizon
- **Plugin / widget SDK** — community-built widgets
- **Localization** — first non-English languages
- **Touch & pen support** — better behavior on Surface and 2-in-1 devices
- **macOS** — exploring; no commitment yet
